Simple sudoku solver covered by a few tests

### How to run App:
Run in terminal `sbt run`<br>
Enter nine lines of the sudoku field using `-` as missing values, e.g.
```
-2--7-43-
-513--2-6
-7---6--1
6-----7-9
73-5---1-
--9-64--8
568-271-3
3-2-1----
--7-8352-
```


### How to run tests:
Run in terminal `sbt test`
